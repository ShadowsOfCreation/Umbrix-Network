# Umbrix Network Roadmap

Umbrix Network is currently in the early planning phase. This roadmap outlines the intended development stages of the platform.

## Phase 1 – Foundation [Status: Completed]
- Establish the Umbrix community
- Define platform vision and architecture
- Design modular system concept
- Plan core platform components

## Phase 2 – Core Platform [Status: In Progress]
- Build initial desktop app shell ✅
- Establish workspace/monorepo structure ✅
- Create shared module contracts ✅
- Define module manifest format ✅
- Implement manifest discovery from filesystem ✅
- Implement module registry ✅
- Render built-in module pages from registry ✅
- Design module permission model
- Design future user/account permission system

## Phase 3 – Initial Modules [Status: In Progress]
- Create Chat module placeholder UI ✅
- Create Projects module placeholder UI ✅
- Add Knowledge module
- Add Documentation module
- Define module lifecycle design
- Define module API surface
- Define background service model

## Phase 4 – Runtime & Integration [Status: Not Started]
- Implement module runtime lifecycle
- Add module enable/disable support
- Add module settings support
- Add module permission enforcement
- Add background service support
- Improve built-in module loading flow

## Phase 5 – Prototype [Status: Not Started]
- Internal testing
- Feature integration
- Early community feedback

## Phase 6 – Public Alpha [Status: Not Started]
- First public release
- Community testing
- Platform iteration
